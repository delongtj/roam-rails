# README

## Overview

Back-end API for Roam trip planning app. Handles authentication and RESTful API via JSON Web Tokens.

## Requirements

* Ruby 2.6.3
* PostgreSQL 9.6
* Rails 5.2.3

## Setup

* Clone to local machine
* `bundle install`
* `rails db:setup`
* `rails s` to start server on port 3000
